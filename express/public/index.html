Why Choose Express?

Express provides a thin layer of fundamental web application features without obscuring Node.js features.

⚡

Fast & Lightweight

Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

🔌

Middleware

Use middleware to handle requests, modify response objects, and perform various tasks in your application's request-response cycle.

🛣️

Routing

Define routes with different HTTP methods that respond to client requests with elegant route handling.

🔧

Extensible

Extend functionality with a vast ecosystem of middleware packages to handle everything from authentication to compression.

🚀

Performance

Built on Node.js, Express leverages its non-blocking I/O model to handle thousands of concurrent connections efficiently.

Community

Simple, Yet Powerful

Express makes it easy to create robust web applications and APIs with just a few lines of code.

// A simple Express server
const express = require('express');
const app = express();
const port = process.env.PORT || 3000;

// Middleware example
app.use((req, res, next) => {
  console.log(`${req.method} ${req.url}`);
  next();
});

// Route handling
app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.get('/api/users', (req, res) => {
  res.json([
    { id: 1, name: 'John Doe' },
    { id: 2, name: 'Jane Smith' }
  ]);
});

// Start the server
app.listen(port, () => {
  console.log(`Server running at http://localhost:${port}`);
});

Getting Started

Start building with Express in minutes

1

Install Node.js

Make sure you have Node.js installed on your system. Download it from nodejs.org.

2

Create a new project

Create a new directory for your project and initialize it with npm:

$ mkdir my-express-app
$ cd my-express-app
$ npm init -y
3

Install Express

Add Express as a dependency to your project:

$ npm install express
4

Create your app

Create an index.js file and add your Express application code:

const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(port, () => {
  console.log(`App listening at http://localhost:${port}`);
});
5

Run your app

Start your Express application:

$ node index.js

Visit http://localhost:3000 in your browser to see your app in action!
